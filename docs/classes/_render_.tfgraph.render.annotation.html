Class Annotation

A class for rendering annotation object which contains label about the node embedded as annotation, type of annotation and the location of both the annotation's node and edge.

Annotation objects include embedded constants, embedded summary, and edge shortcuts.

      The underlying node this annotation points to.

      The render information for the underlying node this annotation points to. This can be null if the annotation denotes an embedding (constant, summary), in which case we use the node property.
