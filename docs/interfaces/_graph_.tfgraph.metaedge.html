Optional inbound

inbound: boolean
  • Defined in graph.ts:477

Whether this edge represents a relationship that is inbound (or outbound) to the object which contains this information. For example, in a Metanode's bridgegraph, each edge connects an immediate child to something outside the Metanode. If the destination of the edge is inside the Metanode, then its inbound property should be true. If the destination is outside the Metanode, then its inbound property should be false.

The property is optional because not all edges can be described as inbound/outbound. For example, in a Metanode's metagraph, all of the edges connect immediate children of the Metanode. None should have an inbound property, or they should be null/undefined.
